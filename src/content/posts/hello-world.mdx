## 它是怎么搭起来的

这个博客并不是用现成主题套出来的，而是从空目录一行行写起来的。技术栈如下：

| 部分 | 选择 |
| --- | --- |
| 框架 | Next.js 16（App Router） |
| 语言 | TypeScript |
| 样式 | Tailwind CSS v4 |
| 内容 | MDX 文件 + gray-matter 解析 frontmatter |
| 高亮 | Shiki |
| 部署 | Vercel |

写一篇文章的方式很简单：在 `src/content/posts/` 下放一个 `.mdx` 文件，
带上 frontmatter（title / date / tags 等），它就会自动出现在文章列表和详情页。

```ts
// src/content/posts/example.mdx
---
title: 示例
date: 2026-05-23
---

写正文……
```

## 接下来要做什么

这是 v0.1 的最小版本。后续的路线图：

1. **v0.2** — 标签页 / 全文搜索 / RSS / sitemap / OG 图像

4. **v0.5** — 邮件订阅 + 自托管访客统计
5. **v1.0** — 多语言 / 付费墙 / AI 摘要 / AI 问答 / 留言 / 友链 / 相册
